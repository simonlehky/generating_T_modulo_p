# Generating transformation matrix T mod p

This repository contains Python scripts for mathematical and statistical analysis, including density and cardinality plotting, invertible block operations, sampling with rejection, and triangular form computations. 

## Project Structure

- `cardinality_plot.py` — Plot cardinality-related data.
- `density_plot.py` — Generate density plots.
- `density_plot_refine.py` — Refined density plotting: Use this to generate figures in the paper.
- `ivertible_block.py` — Operations on invertible blocks.
- `sampling_rejection.py` — Sampling using rejection methods.
- `triangular_form.py` — Triangular form computations.
- `fig/` — Output figures and plots.

## Requirements

Install dependencies with:

```bash
pip install -r requirements.txt
```

## Usage

Activate your virtual environment and run any script, for example:

```bash
python density_plot_refine.py
```
